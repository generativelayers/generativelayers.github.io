How it Works

Request path

1. Agent asks for external support.
↓
2. Adapter creates a Generative Layers request.
↓
3. Governance checks policy, context, limits, and audit requirements.
↓
4. Provider calls an LLM, tool, API, service, or module.
↓
5. Result returns with status, content, metadata, and warnings.
↓
6. Agent decides what to do with it.

Important rule

The framework returns a result. It does not silently change beliefs, plans, intentions, or actions.
