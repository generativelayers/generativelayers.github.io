Welcome to Generative Layers

A Java framework for governed generative resource layers around agent-oriented systems.

Generative Layers focuses on BDI-first and agent-oriented systems where reasoning semantics matter. Its purpose is to support controlled interaction between agent programs and external generative resources such as LLMs, tools, APIs, services, and assessment modules.

Project purpose

The project exists to keep a clear boundary between agent reasoning and external generation. Generated material should be handled as candidate external material, not as automatic belief, intention, plan, or action.

This makes the project relevant to academic MAS research and to engineers building agent systems that need governance, auditability, and explicit control over external resource use.

Intended audience

Researchers

For studying semantic separation, governance, and LLM/tool use in agent-oriented systems.

Engineers

For building Java-based framework infrastructure around agent platforms and external resources.

Agent platform users

For experiments involving Jason, ASTRA, JaCaMo, CArtAgO, and related systems.

Adapter contributors

For connecting other BDI languages or MAS frameworks through explicit integration boundaries.

Target Agent Platforms

The first research direction is BDI-first, using established agent programming and MAS platforms as evaluation targets.

Jason

AgentSpeak-style BDI reference target.

ASTRA

Practical Java-backed agent programming target.

JaCaMo

MAS platform combining agents, environments, and organisations.

CArtAgO

Artifact and environment framework relevant to shared resources.

Project origin

Generative Layers began as MSc thesis-oriented research in Advanced Software Engineering at University College Dublin.
