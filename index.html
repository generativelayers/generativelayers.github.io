Welcome to Generative Layers

Documentation for a governed generative agent framework.

Generative Layers is framework infrastructure for agent systems that need controlled access to external resources, including models, tools, services, and APIs.

The project separates agent-side decisions from external resource calls so that interaction can be checked, logged, and handled through explicit framework boundaries.

Project origin

Generative Layers started as thesis-oriented research at University College Dublin for the MSc in Advanced Software Engineering. The work investigates how governance, decision constraints, and external generative capabilities can be layered around agent systems without collapsing those concerns into the agent's internal reasoning process.
