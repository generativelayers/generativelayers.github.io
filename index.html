Early research framework

Generative Layers

A Java framework for governed generative resource layers around agent-oriented systems.

Purpose

Generative Layers keeps external generation separate from agent reasoning.

LLM, tool, API, and service outputs are returned as candidate material. The host agent platform decides whether to adopt, verify, ignore, or reject them.

Target platforms

Jason

AgentSpeak-style BDI target.

ASTRA

Java-backed agent programming target.

JaCaMo

Agents, environments, and organisations.

CArtAgO

Artifacts and environment resources.

Academic origin

Generative Layers began as MSc thesis-oriented research in Advanced Software Engineering.
