Getting Started

Early development

The project is currently in early research and framework design.

Current route

Start with the framework repository, then use the examples repository to follow platform-specific integration work as it matures.

Open framework repoOpen examples repo

Installation status

Maven and Gradle installation instructions will be published with the first release. Until then, the website should be treated as research and framework documentation, not as a released package manual.

Expected engineering path

  • Java framework core.
  • Governance and audit abstractions.
  • Provider abstraction for LLMs, tools, APIs, and services.
  • Platform adapters for agent-oriented systems.
  • Runnable examples and repeatable evaluation cases.
