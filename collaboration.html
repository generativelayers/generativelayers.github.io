Collaboration

Generative Layers welcomes discussion with researchers and engineers working on BDI languages, MAS frameworks, and agent-oriented platforms.

External adapters

If you are interested in connecting Generative Layers to another BDI language, MAS framework, or agent-oriented platform, contact the project team.

For adapter or integration discussions, contact: info [at] generativelayers.com.

Please include the target language or framework, the intended integration idea, and any public repository, prototype, or technical notes that help explain the proposal.

Useful information to include

  • Target language, framework, or runtime.
  • Reason the integration is relevant to governed generative resource layers.
  • Existing code, repository link, prototype, or technical notes if available.
  • Expected collaboration type, such as adapter contribution, example project, research prototype, or joint experiment.

Contribution focus

Strong contributions should preserve the central design rule: generated material is candidate external material and should not silently become agent beliefs, intentions, plans, or actions.
