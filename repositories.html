Companion Tools

To support agent development under transactional governance constraints, lightweight static analysis tools were built to audit ASTRA and Jason codebase specifications. These companion tools perform real-time verification of AgentSpeak syntax and BDI reasoning rules.

astra-diagnostics

VS Code

Provides static analysis, syntactic validation, and real-time schema linting for ASTRA agent source files, mapping 121 checks across 31 rulesets.

MIT GitLab

jason-diagnostics

VS Code

Provides real-time AST linting, AgentSpeak semantics checking, and BDI belief/goal flow validation for Jason (.asl) source files across 135 checks.

MIT GitLab
