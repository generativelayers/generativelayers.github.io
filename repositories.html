Repositories

Generative Layers is organised around the framework, examples, and documentation repositories.

framework

Main Java framework repository for Generative Layers.

examples

Example projects for Jason, ASTRA, JaCaMo, and CArtAgO integrations.

docs

Documentation source, guides, reference material, and research notes.

Repository purpose

The repositories separate framework implementation, runnable examples, and documentation so that the academic research material and Java engineering work can mature without being mixed together.
