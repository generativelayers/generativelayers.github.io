Research

Governed generative resource layers for agent-oriented systems.

The Problem

Agent programs need access to LLMs, tools, APIs, and services without allowing those outputs to silently become beliefs, intentions, plans, or actions.

The Contribution

A Java framework boundary between agent execution and external generative resource use.

Research Questions

The framework addresses three foundational questions at the intersection of BDI theory and generative models:

RQ1

State Isolation

How can generated outputs be exposed to BDI agents without directly modifying agent state?

RQ2

Auditable Boundaries

What governance boundary is needed for auditable resource access?

RQ3

Cross-Platform Parity

Can the same resource-layer model work across Jason, ASTRA, and related MAS frameworks?
