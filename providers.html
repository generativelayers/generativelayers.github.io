Providers

Configure LLM providers for your agents — built-in services, custom endpoints, and runtime switching.

Overview

Generative Layers supports multiple LLM providers out of the box. Call bind(agent, provider, model, config) to activate a provider — no framework changes needed. For unlisted providers, also set the endpoint and API key environment variable.

Usage Examples
Cerebras (Recommended Free Option)
High-speed wafer-scale inference. Free tier with no credit card required. Best for development and testing.
gl.bind("agent1", "cerebras", "gpt-oss-120b", "");
gl.bind("agent1", "cerebras", "gpt-oss-120b", "", Bid);
bind("agent1", "cerebras", "gpt-oss-120b", "", Bid);
Groq
Ultra-fast inference with generous free tier.
gl.bind("agent1", "groq", "llama-3.3-70b-versatile", "");
gl.bind("agent1", "groq", "llama-3.3-70b-versatile", "", Bid);
bind("agent1", "groq", "llama-3.3-70b-versatile", "", Bid);
Gemini (Native Google API)
Gemini uses Google's native API. Just call bind(agent, "gemini", model, "").

gl.bind("agent1", "gemini", "gemini-2.5-flash", "", Bid);
bind("agent1", "gemini", "gemini-2.5-flash", "", Bid);
Using Any Custom Provider (Zero Code Changes)

Any service implementing the Chat Completions API format works immediately — no framework update required. Just provide the endpoint, model name, and API key environment variable. This includes Together AI, Fireworks, Ollama (local), LM Studio, Azure OpenAI, and any future provider. 

// Any OpenAI-compatible endpoint works
gl.bind("agent1", "chatcompletions", "your-model-name", "endpoint=https://api.your-provider.com/v1/chat/completions;apiKeyEnv=YOUR_API_KEY");
// Any OpenAI-compatible endpoint works
gl.bind("agent1", "chatcompletions", "your-model-name", "endpoint=https://api.your-provider.com/v1/chat/completions;apiKeyEnv=YOUR_API_KEY", Bid);
// Any OpenAI-compatible endpoint works
bind("agent1", "chatcompletions", "your-model-name", "endpoint=https://api.your-provider.com/v1/chat/completions;apiKeyEnv=YOUR_API_KEY", Bid);
Example: xAI Grok, Mistral, Together AI
Set the API key as an environment variable (e.g. export GROK_API_KEY="xai-..."), then bind: 
// xAI Grok
gl.bind("agent1", "chatcompletions", "grok-2", "endpoint=https://api.x.ai/v1/chat/completions;apiKeyEnv=GROK_API_KEY");

// Mistral
gl.bind("agent1", "chatcompletions", "mistral-large-latest", "endpoint=https://api.mistral.ai/v1/chat/completions;apiKeyEnv=MISTRAL_API_KEY");
Switching Providers at Runtime

Providers can be switched at any point during execution. This enables patterns like Cross-LLM Verification, where one LLM generates and another verifies. 

// Bind to Gemini, then call
!classified(gl.call(gl.bind("agent1", "gemini", "gemini-2.5-flash", ""), "classify", "llm.answer", "ANSWER", "Classify: apple", "label", ""));

// Bind to Cerebras, then verify
!verified(gl.call(gl.bind("agent1", "cerebras", "gpt-oss-120b", ""), "verify", "llm.answer", "ANSWER", "Is apple = fruit correct?", "verdict", ""));
Auto-Detection
If no provider is specified in bind(), the framework auto-detects by checking which API key environment variables are set. Priority order: CEREBRAS_API_KEY → GROQ_API_KEY → GEMINI_API_KEY → OPENAI_API_KEY → DEEPSEEK_API_KEY. Falls back to a deterministic fake provider for testing without any API key.
